1) Все классы для этой задачи должны располагаться в папке "src/main/java"
2) Создвть класс Animal, описывающий общий тип животных
3) Класс должен содержать следующие поля:
    noOfLegs - кол-во ног животного
    name - название животного
    vegetarian - это поле должно дать сведения, является ли животное травоядным или нет.
    (Всего может быть 2 варианта для этого поля)
    weight - вес животного
    height - высота животного
    age - возраст животного
4) Класс должен содержать пустой конструктор(без параметров) и конструктор с параметрами для всех полей класса.
5) Переопределить toString, чтобы он возвращал имя, возраст, высоту и длину животного.
6) Для всех полей добавить геттеры/сеттеры.
Правила написания класса:
сначала идут все поля класса, потом пустой конструктов, конструктор с параметрами, потом геттеры/сеттеры и последним метод toString()
